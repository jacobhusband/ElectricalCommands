# Control Schedule commands

Run `CSCHED` (or `CONTROLSCHEDULE`) in AutoCAD to open the modeless editor.

- **Load Selected** reads an existing generated Control Schedule back into the editor.
- **Place New Table** prompts for an insertion point and creates a six-column AutoCAD table.
- **Update Linked / Selected** replaces the linked table in place, preserving its layer, rotation, table style, and insertion point.
- **Reset Defaults** restores the reference Wattstopper schedule rows and notes.

The generated table stores its editable values directly in its cells, so it remains portable with the DWG and can be loaded again in a later AutoCAD session.
